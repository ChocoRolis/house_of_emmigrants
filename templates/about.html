About Our Platform

Discover the story behind our interactive data exploration tool for Swedish migration history

Back to Home

This web application serves as an interactive data exploration and visualization tool for the House of Emigrants collection at Kulturparken Småland. It enables researchers, historians, and the general public to discover patterns in 19th- and 20th-century Swedish migration to North America through dynamic charts, maps, and keyword analyses. As new archival records and interviews are processed, the platform automatically updates, ensuring up-to-date insights into departure trends, popular migration motives, and individual narratives.

Key Features

Interactive Dashboard

View temporal trends in departures, top migration keywords, and geographic distributions via line, column, and pie charts.

Secure Authentication

Administrative users can log in to upload and manage text and multimedia files for automated data extraction.

Automated Processing

Uploaded text files are parsed to extract dates, places, names, and keywords, which are validated and stored in a normalized PostgreSQL database.

Responsive Design

Built with Bootstrap and modern web standards for seamless use across devices.

API-Driven

A Flask-based REST API serves processed data for efficient front-end updates.

Scalable Architecture

Designed for cloud deployment and horizontal scaling as data volume grows.
